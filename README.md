# Personal Portfolio Website

My personal website showcasing my journey, tech stack, and projects as a Game Developer and Video Editor. Built with a focus on clean code, performance, and minimal design.

## Tech Stack

- **Frontend:** React / Vite, TypeScript, Tailwind CSS
- **UI Components:** shadcn/ui
- **Animation:** Framer Motion (Optimized with Scroll-Driven animations)

## Features

- **Global Transitions:** Smooth page-wide initial fade-in load animations.
- **Scroll Reveal:** Elegant slide-up components triggered natively on scroll.
- **Accessible Modals:** Image preview modal component for showcasing credentials and screenshots cleanly.
- **Copy-to-Clipboard:** Direct, seamless contact integration for Discord.

## Project Setup

To run this project locally, ensure you have [Node.js](https://nodejs.org/) installed, then follow these steps:

1. Clone the repository:
   ```bash
   git clone [https://github.com/fyrddotdev/Fyrd.dev-s-Personal-Page.git](https://github.com/fyrddotdev/Fyrd.dev-s-Personal-Page.git)
   ```

2. Install Dependencies:
   ```bash
   npm install
   ```
3. Start the local development server:
   ```bash
   npm run dev
   ```
