# Fyrd.dev_'s Personal Portfolio Website

My first personal website showcasing my journey, tech stack, and projects as a programmer. Built with a focus on clean code and minimal design.

## Tech Stack

- **Frontend:** React / Vite, TypeScript, Tailwind CSS
- **UI Components:** shadcn/ui
- **Animation:** Framer Motion

## Project Setup

To run this project locally, ensure you have [Node.js](https://nodejs.org/) installed, then follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/fyrddotdev/Fyrd.dev-s-Personal-Page.git
   ```

2. Install Dependencies:
   ```bash
   npm install
   ```
3. Start the local development server:
   ```bash
   npm run dev
   ```
